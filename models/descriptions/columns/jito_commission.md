{% docs jito_commission %}
The commission rate (in basis points) that the validator charges for Jito MEV services. This represents the percentage of MEV rewards that the validator retains, with the remainder distributed to delegators. For example, a value of 200 represents a 2% commission rate. Only applicable for validators that support Jito (is_jito = TRUE).

Data type: NUMBER (basis points)
